# Atlas Growth Partners Website

Marketing homepage for atlasgrowthpartners.io built with:

- Next.js 14 (App Router)
- Tailwind CSS
- Framer Motion

## Local Development

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Brand Tokens

- atlas-blue: `#4BC8E8`
- atlas-dark (background): `#0A0A0A`
- atlas-surface: `#1A1A1A`
